# Dallas Brown Portfolio

Static GitHub Pages portfolio for Dallas Brown, an Information Systems student and early-career analytics professional.

## Local preview

From the repository root, run:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deployment

The site is plain HTML, CSS, and JavaScript. GitHub Pages publishes the `main` branch directly; no build command or server is required.

## Featured work

- HabitFlow - NFC-powered habit tracking, a FastAPI backend, and an installable analytics dashboard.
- Lyfe - 2nd-place Microsoft Intern Hackathon 2026 project.
